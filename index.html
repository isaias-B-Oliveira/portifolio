Tecnologias

tecnologias que conheço

  • Logica de programação

    Lógica de programação é o processo de projetar e escrever algoritmos de maneira clara e eficiente para resolver problemas computacionais.

  • HTML

    linguagem de marcação utilizada para criar a estrutura e o conteúdo . É a base fundamental para a criação de sites e é interpretada pelos navegadores da web para exibir o conteúdo ao usuário.

  • CSS

    é uma linguagem de estilo utilizada para controlar a apresentação visual e o layout de documentos HTML.é possível definir cores, fontes, tamanhos, margens, posicionamento e outros aspectos visuais de um documento web.

  • JavaScript

    JavaScript é uma linguagem de programação de alto nível, dinâmica e interpretada, amplamente utilizada para desenvolvimento web.

  • React

    React é uma biblioteca JavaScript. Ela permite construir componentes reutilizáveis que atualizam e renderizam automaticamente quando os dados mudam, tornando o desenvolvimento de aplicativos web mais rápido e eficiente.

  • Git

    Git é um sistema de controle de versão distribuído amplamente utilizado para rastrear e gerenciar alterações em projetos de desenvolvimento de software. Ele foi desenvolvido por Linus Torvalds em 2005
